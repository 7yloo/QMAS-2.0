# Q-MAS 2.0 Usage Guide

## Command Line Options

| Option | Description | Default |
|--------|-------------|---------|
| `--epochs` | Number of epochs | 10 |
| `--agents` | Number of agents | 100 |
| `--output` | Output file name | results.csv |
